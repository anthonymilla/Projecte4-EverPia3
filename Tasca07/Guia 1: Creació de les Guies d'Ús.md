#### Com gestionar funcions clau (transferència d'arxius, canvi de pantalla, reinici remot).

- La security (seguretat), el Firewall.

![](img/38.png)

La segona opció serveix per gestionar i controlar l'ordinador remot a un nivell més profund, com si estiguessis físicament davant d'ell.

"Send Ctrl + Alt + Del" serveix per enviar la combinació de tecles Ctrl+Alt+Supr a un ordinador remot.

Aquesta combinació de tecles és molt útil per:
- Accedir al Gestor de tasques: Permet veure els programes que s'estan executant, tancar aplicacions que no responen o veure l'ús de recursos del sistema.
- Bloquejar o tancar la sessió: Permet canviar d'usuari, tancar la sessió actual o reiniciar/apagar l'ordinador.
- Accedir a opcions de seguretat: En alguns sistemes, pot donar accés a opcions com canviar la contrasenya.

En el nostre cas no es compatible amb el sistema operatiu remot (This functionality is not supported for the remote operating system), el sistema operatiu no ho permet.

![](img/40.png)

I la quarta opció ja és el reboot (reiniciar). El reboot serveix per reiniciar un ordinador o sistema. Això doncs ajuda a aplicar actualitzacions, solucionar problemes o restaurar configuracions.

![](img/41.png)

Ara altres opcions que hi han. A les opcions de Communicate (Comunicat), hi ha:
- Whiteboard (Pissarra): Permet obrir una pissarra per dibuixar o escriure.
- Show participants (Mostra participants): Mostra la llista de persones connectades a la sessió.
- Computer sound (So de l'ordinador): Permet compartir o silenciar l'àudio de l'ordinador.
- Augment session (Augmenta sessió): Ofereix opcions addicionals per millorar la sessió (com gravar sessió per exemple). 

![](img/47.png)

A les opcions de View (Veure), hi ha:
- Scaling (Escalat): Ajusta la mida de la pantalla remota perquè s'adapti millor al teu monitor.
- Quality (Qualitat): Permet configurar la qualitat de la imatge de la pantalla remota (per exemple, millor velocitat o millor detall).
- View monitors (Veure monitors): Selecciona quin monitor remot vols veure si n'hi ha diversos connectats.
- Screen resolution (Resolució de pantalla): Canvia la resolució de la pantalla de l'ordinador remot.
- Hide wallpaper (Amagar fons de pantalla): Desactiva el fons d'escriptori de l'equip remot per millorar el rendiment i la velocitat.
- Show remote cursor (Mostrar cursor remot): Mostra el cursor del ratolí de l'ordinador remot a la teva pantalla.
- Refresh screen (Actualitzar pantalla): Força una actualització manual de la imatge de la pantalla remota si no es veu correctament.
- Select screen content (Seleccionar contingut de pantalla): Permet triar quines parts específiques de la pantalla remota vols compartir o veure.
- Show sessions in tabs (Mostrar sessions en pestanyes): Organitza les diferents sessions remotes que tinguis obertes en pestanyes separades per facilitar la navegació.

![](img/48.png)

A les opcions de Actions (Accions), hi ha:
- Leave note (Deixar nota): Permet afegir una nota escrita per a tu o per a l'usuari remot.
- Lock (Bloquejar): Bloqueja la sessió remota, impedint interaccions fins que es desbloquegi.
- Remote reboot (Reinici remot): Reinicia l'ordinador remot.
- Send Ctrl + Alt + Del (Enviar Ctrl + Alt + Supr): Envia la combinació de tecles Ctrl+Alt+Supr a l'ordinador remot.
- Clipboard (Portapapers): Permet sincronitzar el portapapers entre el teu ordinador i el remot.
- Send key combinations (Enviar combinacions de tecles): Envia combinacions de tecles específiques a l'ordinador remot.
- Direct keyboard mode (Mode de teclat directe): Activa un mode d'entrada de teclat més directe per a una millor resposta.
- Information (Informació): Mostra informació detallada sobre la sessió remota activa.
- Remote update (Actualització remota): Permet actualitzar TeamViewer a l'equip remot a la versió més recent.

![](img/49.png)

#### Bones pràctiques de seguretat (p. ex., tancar sempre la sessió, no desar contrasenyes de clients).
Bones pràctiques de seguretat serien mantenir sempre la sessió tancada quan no s’utilitza, perquè aixís si deixes l’ordinador sense vigilància ningú pugui entrar al teu compte, utilitzar contrasenyes segures, perquè ningun intrús pugui entrar amb facilitat. També no deixar informació sensible a la vista, per evitar que algú vegi la informació sense voler (un intrús) i sempre assegurar-se que només les persones qué doncs són fiables accedeixen a les dades, perquè no hi hagi qualsevol problema en general (persones no fiables que poden fer-te perdre dades importants d’algú, intrusos).
