# ACCÉS REMOT. ESCRIPTORI REMOT (RDP)

Primer de tot posem Xarxa NAT al Adaptador 1 a les dues màquines virtuals perquè es puguin veure.

![Primer de tot posem Xarxa NAT al Adaptador 1 a les dues màquines virtuals perquè es puguin veure.](img/Imatge01.png)

Primer anem a Windows a configurar-lo perquè accepti connexions, anem a Configuració, Sistema.

![Primer anem a Windows a configurar-lo perquè accepti connexions, anem a Configuració, Sistema.](img/Imatge02.png)

Ara busquem Escriptori Remot, activem l'opció Escriptori Remot.

![Ara busquem Escriptori Remot, activem l'opció Escriptori Remot.](img/Imatge03.png)

També marquem la casella.

![També marquem la casella.](img/Imatge04.png)

Cliquem on posa Usuaris d'Escriptori remot, ara afegir.

![Cliquem on posa Usuaris d'Escriptori remot, ara afegir.](img/Imatge05.png)

Posem el nom de PC i el nom; user en aquest cas.

![Posem el nom de PC i el nom; user en aquest cas.](img/Imatge06.png)

Acceptem i ja hem fet la configuració Windows.

![Acceptem i ja hem fet la configuració Windows.](img/Imatge07.png)

Ara fem la configuració Zorin. Per això anem a Configuració, després a Sistema i Escriptori Remot.

![Ara fem la configuració Zorin. Per això anem a Configuració, després a Sistema i Escriptori Remot.](img/Imatge08.png)

I posem en “on” tant Compartició d’Escriptori com Control Remot.

![I posem en “on” tant Compartició d’Escriptori com Control Remot.](img/Imatge09.png)

Ara entrem a Remmina.

![Ara entrem a Remmina.](img/Imatge10.png)

Posem l’IP del Windows i donem Enter.

![Posem l’IP del Windows i donem Enter.](img/Imatge11.png)

Posem la següent comanda a Windows per veure l’IP per això.

```
ipconfig
```

![Posem la següent comanda a Windows per veure l’IP per això.](img/Imatge12.png)

Ara Acceptem (Yes).

![Ara Acceptem (Yes).](img/Imatge13.png)

Posem l’username i el password per fer la connexió al Windows.

![Posem l’username i el password per fer la connexió al Windows.](img/Imatge14.png)

Ja estaria fet la connexió Zorin Windows.

![Ja estaria fet la connexió Zorin Windows.](img/Imatge15.png)

Ara anem a Windows a connexió a Escriptori remot. Posem el nom de l’equip i connectem.

![Ara anem a Windows a connexió a Escriptori remot. Posem el nom de l’equip i connectem.](img/Imatge16.png)

Posem el nom d’usuari i contrasenya corresponent i acceptem.

![Posem el nom d’usuari i contrasenya corresponent i acceptem.](img/Imatge17.png)

Diem que si.

![Diem que si.](img/Imatge18.png)

Ja estaria fet la connexió Windows Zorin.

![Ja estaria fet la connexió Windows Zorin.](img/Imatge19.png)
